사용 요금

본 문서는 요금, 플랜 및 무료 사용량에 관해 설명합니다. 모든 요금은 부가세(VAT) 별도 금액입니다.

요금제 개요​

  • 기본 가격 구간에 따라 사용량이 많아질수록 시간당 단가가 낮아집니다.
  • 각 구간별 일반 STT와 스트리밍 STT 모델에 동일한 요율이 적용됩니다.

모델별 적용 대상​

모델 카테고리지원 모델 목록
일반 STTsommers, whisper
스트리밍 STTsommers_ko, sommers_ja, whisper

가격 구간별 요금​

구간월 사용량 (시간)시간당 요금 (원)
T10 ≤ 사용량 ≤ 1,000시간1,000
T21,000 < 사용량 ≤ 10,000시간500
T310,000 < 사용량 ≤ 25,000시간400
T425,000시간 초과300
주의사항
  • 가격 계산시 구간별 누진 과금 방식이 적용됩니다.
  • T3 이상 사용시, 사전 문의 를 주셔야 구간별 요금 적용이 가능합니다.

청구 및 과금 방식​

  1. 집계 단위
    • 오디오 파일 또는 스트리밍 세션당 최소 집계 단위는 10초입니다.
    • 실제 사용 시간이 10초 미만일 경우에도 10초로 집계됩니다.
    • 사용량은 초 단위로 집계한 뒤 시간 단위(초 ÷ 3,600)로 환산합니다.
      • 예: 5초 사용 → 10초로 집계 → 10 ÷ 3,600 ≈ 0.0028시간
      • 예: 95초(1분 35초) 사용 → 95초로 집계 → 95 ÷ 3,600 ≈ 0.0264시간
  2. 파일 기반(STT)
    • 음성 파일 총 길이를 기준으로 집계
    • 다중 채널 옵션 사용 시 채널별 길이 합산
  3. 실시간 스트리밍(STT)
    • 연결된 스트림 오디오 시간을 기준으로 집계
월 2,500시간 사용 시 (계산 예시)
  • T1(0~1,000시간): 1,000시간 × 1,000원 = 1,000,000원
  • T2(1,001~2,500시간): 1,500시간 × 500원 = 750,000원
  • 총 과금액 = 1,000,000원 + 750,000원 = 1,750,000원
  • 부가세 포함 총액 = 1,750,000원 + 10% = 1,925,000원

플랜​

  • 모든 사용자는 가입 즉시 베이직 플랜으로 시작합니다.
  • 카드결제가 아닌 세금계산서 방식으로 결제가 필요한 경우 별도의 계약이 필요합니다.
  • 대용량 사용이 필요한 사용자나 기업의 경우, 아래 정보와 함께 이메일 혹은 채팅 상담 바로가기를 클릭해 주세요.
  • 사전 문의시 T3 이상 구간별 요금 적용이 가능합니다. 대용량 사용시 추가적인 할인 요금이 적용될 수 있습니다.

무료 사용량 안내​

  • 가입 즉시 기본 600분(10시간) 무료 제공
  • 일반 STT와 스트리밍 STT 사용 시간을 합산하여 부여
  • 무료 분 초과분부터 가격 구간별 요금 적용
  • 무료 사용량에 대해서는 별도의 요금이 청구되지 않습니다.
