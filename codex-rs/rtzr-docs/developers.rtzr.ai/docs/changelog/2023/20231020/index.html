2023-10-20

2023년 10월 20일

신규 기능 오픈 및 무료 테스트 시간 정책 변경 안내​

🚀 키워드 부스팅 기능 오픈​

https://developers.rtzr.ai/docs/stt-file/keywords

  • RTZR STT는 100만명이 사용한 자체 서비스 VITO 앱을 통해 수집된 데이터를 중심으로 약 12만 시간 이상의 학습데이터를 학습 시킨 모델입니다.
  • 그렇지만, 우리 회사, 우리 산업에서만 쓰이는 단어는 잘 전사하지 못한다는 피드백이 있었는데요. 이런 불편함을 해결하기 위해서 오픈된 키워드 부스팅 기능!!
  • 더 잘 알아듣기 바라는 100개의 단어 뭉치를 API 호출 시 함께 전달하면 해당 단어들에 대한 인식율이 확 높아집니다.
  • 우리의 특수한 용어는 전사가 잘 되지 않아서, 서비스 도입을 망설이거나 불편하게 느끼셨던 분들은 바로 "키워드 부스팅 기능"을 확인해보세요.

💬 word timestamp 기능​

https://developers.rtzr.ai/docs/stt-file/word_timestamp/

  • 전사된 텍스트를 원래 오디오와 정확히 정렬시킬 필요가 있을 때 특히 유용한 word timestamp 기능을 제공합니다.
  • 특히 STT를 이용한 자막 제작 사용에 유용합니다.

🏹 도메인 선택 기능​

https://developers.rtzr.ai/docs/stt-file/domain/

  • 도메인 설정은 STT API를 이용할 때 특정한 상황이나 용도에 따라 음성인식 모델을 선택할 수 있게 해주는 기능입니다.
  • 전화통화를 위주로 STT를 이용해주시는 고객분들은 이 설정을 CALL 로 입력하여 사용하시면 더 좋은 성능을 경험하실 수 있습니다.
