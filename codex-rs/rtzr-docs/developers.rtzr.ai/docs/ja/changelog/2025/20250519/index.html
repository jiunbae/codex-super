2025-05-19

2025年5月19日

🚀 New Feature​

일반STT​

  • 많은 기업과 개인고객들이 요청해주셨던 RTZR Whisper 모델에 대해서 이제 별도로 컨택 없이 사용가능합니다. 동시처리 제한은 여전히 있기 때문에 동시에 여러파일을 POST요청하기 위해서는 컨택이 필요합니다.
  • Whisper 모델 사용시 자동으로 언어 감지하는 기능이 추가되었습니다. 언어 감지 기능을 참고하세요.

스트리밍STT​

  • Domain 기능이 추가되었습니다. "CALL" domain 선택시 전사 품질이 콜센터와 같은 환경에서 최적화되며, "MEETING" 사용시 회의실과 같은 환경에서 최적화된 텍스트가 만들어집니다.

🐛 Bug Fix​

  • 일반STT 사용시 GET 호출을 POST 이후 20ms 이후 연속적으로 호출하는 경우 간헐적으로 404 발생하던 버그가 수정되었습니다.

📝 Documentation​

  • 스트리밍STT의 gRPC와 WebSocket의 공통 부분이 좀 더 알아보기 쉽게 변경 되었습니다. 공통 DecoderConfig/Parameter 정보
  • 구간별 가격 정책이 업데이트 되었습니다. 최대 300원, 일반적인 기업 고객의 경우 시간당 400원까지의 STT요금제를 사용 가능합니다. 할인구간에 대한 적용은 모델별로 별도 적용됩니다.
