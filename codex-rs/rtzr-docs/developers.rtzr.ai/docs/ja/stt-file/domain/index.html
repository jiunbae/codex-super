ドメイン設定

利用環境に合わせてモデルを最適化するための設定です。domain に GENERAL または CALL を指定します。

  • GENERAL（既定）: 汎用ドメイン。多様なシーンに対応。
  • CALL: 通話・コールセンター音声に最適化。

使い方​

POST のリクエストボディで domain を GENERAL または CALL に設定します。

POST example: Example

Unified example (presets): Example

注意

「CALL」は通話録音に特化しているため、その他の状況では性能が低下する場合があります。
